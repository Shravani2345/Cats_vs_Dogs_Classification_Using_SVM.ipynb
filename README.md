# Cats vs Dogs Image Classification using Support Vector Machine (SVM)

## Objective
Develop an image classification model using Support Vector Machine (SVM) to classify images of cats and dogs.

## Dataset
Microsoft Cats and Dogs Dataset (Kaggle)

https://www.kaggle.com/datasets/shaunthesheep/microsoft-catsvsdogs-dataset

## Technologies Used
- Python
- Google Colab
- OpenCV
- NumPy
- Scikit-learn
- Matplotlib

## Machine Learning Algorithm
Support Vector Machine (SVM)

## Workflow
- Load the image dataset
- Convert images to grayscale
- Resize images to 64×64 pixels
- Flatten images into feature vectors
- Split data into training and testing sets
- Train an SVM classifier
- Evaluate the model using accuracy, confusion matrix, and classification report

## Results
- Successfully classified cat and dog images using SVM.
- Evaluated model performance on the test dataset.
